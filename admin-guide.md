# Hướng Dẫn Sử Dụng Admin Dashboard

## 1. Đăng nhập
- Truy cập trang `/admin`.
- Đăng nhập với vai trò `admin` bằng email: `admin@elitestore.com` và mật khẩu: `admin123`.

## 2. Quản lý sản phẩm
- Truy cập `/admin/products` để thêm, chỉnh sửa hoặc xóa sản phẩm.
- Sử dụng form để nhập thông tin sản phẩm.

## 3. Quản lý danh mục
- Truy cập `/admin/categories` để quản lý danh mục sản phẩm.

## 4. Quản lý đơn hàng
- Truy cập `/admin/orders` để xem và xử lý đơn hàng.

## 5. Quản lý khách hàng
- Truy cập `/admin/customers` để xem thông tin khách hàng.

## 6. Phân tích dữ liệu
- Truy cập `/admin/analytics` để xem báo cáo và biểu đồ phân tích.

## 7. Công cụ marketing
- Truy cập `/admin/marketing` để quản lý chiến dịch marketing.

## 8. Cài đặt
- Truy cập `/admin/settings` để cấu hình các tùy chọn hệ thống.
